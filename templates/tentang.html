Tentang CryptoSentiment

Platform analisis sentimen dan harga cryptocurrency yang menggabungkan data real-time, Natural Language Processing (NLP), dan visualisasi interaktif.

Apa itu CryptoSentiment?

CryptoSentiment adalah aplikasi web yang memantau pergerakan harga kripto secara real-time dan menganalisis sentimen berita menggunakan AI. Aplikasi ini memberikan insight cepat dan akurat untuk membantu pengguna memahami mood pasar hanya dalam beberapa detik.

Teknologi yang Digunakan

  • Flask Web Framework — Backend & Routing
  • CoinGecko API — Data harga crypto real-time
  • NewsData.io API — Berita crypto global
  • FinBERT — Model AI untuk analisis sentimen yang lebih akurat
  • Plotly — Visualisasi chart interaktif
  • TailwindCSS — UI modern dan responsive

Analisis Sentimen dengan FinBERT

FinBERT adalah model NLP berbasis BERT yang di-training khusus untuk menganalisis teks keuangan. Dibandingkan TextBlob atau model umum, FinBERT jauh lebih presisi dalam memahami konteks berita finansial seperti regulasi, laporan ekonomi, dan pergerakan pasar.

Bagaimana CryptoSentiment Menggunakan FinBERT?

  • Mengambil berita terbaru dari NewsData.io
  • Mengirim judul berita ke API FinBERT
  • FinBERT mengembalikan label: Positive, Negative, atau Neutral
  • Skor probabilitas disimpan & divisualisasikan dalam dashboard

Dengan menggunakan FinBERT, aplikasi mampu menyajikan profil sentimen pasar yang lebih akurat dan profesional.
